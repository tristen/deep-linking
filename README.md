## Deep linking
A stripped down version of the Dynamic Deep-Linking and Highlighting UI, [Emphasis](https://github.com/NYTimes/Emphasis).

### Quick Usage

``` js
var content = document.getElementsByTagName('p');
new DeepLinking(content);
```
